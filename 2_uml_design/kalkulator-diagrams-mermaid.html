Class Diagram — Struktur Kode JavaScript
Cetak biru tiga class utama beserta properti, method, dan relasi antar class.
classDiagram class Calculator { +number operand1 +number operand2 +string operator +string displayValue +string currentState +appendDigit(n) void +setOperator(op) void +calculate() number +clear() void +clearEntry() void } class Display { +HTMLElement mainScreen +HTMLElement subScreen +render(val) void +showExpression(expr) void +showError() void } class EventHandler { +Calculator calculator +bindButtons() void +bindKeyboard() void +handleClick(e) void } Calculator ..> Display : menggunakan EventHandler ..> Calculator : mengontrol

Kode Mermaid.js

classDiagram
    class Calculator {
        +number operand1
        +number operand2
        +string operator
        +string displayValue
        +string currentState
        +appendDigit(n) void
        +setOperator(op) void
        +calculate() number
        +clear() void
        +clearEntry() void
    }

    class Display {
        +HTMLElement mainScreen
        +HTMLElement subScreen
        +render(val) void
        +showExpression(expr) void
        +showError() void
    }

    class EventHandler {
        +Calculator calculator
        +bindButtons() void
        +bindKeyboard() void
        +handleClick(e) void
    }

    Calculator ..> Display : menggunakan
    EventHandler ..> Calculator : mengontrol
